About this Project

In the summer of 2013, I interned as a game designer and developer at NYC-based TreSensa, a tech company with a unique focus: game development for mobile Web browsers, and a suite of associated services for developers, publishers, and brands. Their flagship product: a game development engine called TGE (TreSensa Game Engine), based in JavaScript and HTML5. In the fall of 2014, TreSensa brought me back on as a project consultant to help create and improve documentation for their services, with a focus on TGE.

The documentation was spread across three platforms, and I contributed to all of them: a collection of written articles, a series of interactive (and editable) code demos on the live coding Web site CodePen (where I also served as the editor and curator of TreSensa's profile), and a GitHub repository of larger code samples.

I wrote several articles on topics such as animation and integrating social features, which you can read here, and I encourage you to do so because they are pretty good. The "Tween" article in particular is a tour de force.
